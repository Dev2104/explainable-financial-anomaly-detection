# Evaluating Explainable AI-Based Anomaly Detection Models for Financial Trading Surveillance

This MSc Data Science dissertation investigates unusual trading behaviour surrounding quarterly earnings announcements using unsupervised anomaly detection models and Explainable AI.

## Scope

Companies:

* AAPL
* MSFT
* NVDA
* AMZN
* TSLA

Time period:

* January 2021 to December 2025

Data sources:

* Yahoo Finance
* Nasdaq Earnings Calendar
* Investing.com Earnings Calendar

## Models

* Isolation Forest
* One-Class SVM
* Local Outlier Factor

## Evaluation Strategy

This project uses unsupervised event-based evaluation. It does not attempt to detect fraud, insider trading, or market manipulation.

## Explainability

SHAP is used to explain the most suitable anomaly detection model.
